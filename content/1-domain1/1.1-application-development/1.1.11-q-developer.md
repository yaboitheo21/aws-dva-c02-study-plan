## Use Amazon Q Developer to Assist with Development

### Tổng quan

Amazon Q Developer là AI-powered coding assistant của AWS, tích hợp trực tiếp trong IDE.

### Tính năng chính

| Feature | Mô tả |
|---------|-------|
| Code Generation | Sinh code từ comments hoặc natural language |
| Code Completion | Gợi ý code real-time khi đang viết |
| Code Explanation | Giải thích đoạn code đang chọn |
| Code Transformation | Upgrade Java versions, modernize code |
| Security Scanning | Phát hiện vulnerabilities trong code |
| Chat | Hỏi đáp về AWS services, coding, debugging |

### Security Scanning

- Quét code để tìm security vulnerabilities
- Hỗ trợ nhiều ngôn ngữ: Python, Java, JavaScript, TypeScript, C#, Go, etc.
- Phát hiện: hardcoded credentials, SQL injection, XSS, insecure crypto
- Đề xuất remediation steps

### IDE Integration

- VS Code
- JetBrains IDEs (IntelliJ, PyCharm, etc.)
- AWS Cloud9
- AWS Lambda Console

### Pricing Tiers

| Tier | Mô tả |
|------|-------|
| Free | Code completion, chat, security scanning (limited) |
| Pro | Unlimited usage, admin controls, organizational policies |

### Use Cases trong Exam

- Tăng developer productivity
- Code review và security best practices
- Giải thích AWS service configurations
- Troubleshoot errors và exceptions
