### Tham khảo

# AWS Services trong phạm vi thi DVA-C02

Danh sách đầy đủ các AWS services có thể xuất hiện trong kỳ thi.

| Category | Services |
|----------|----------|
| **Analytics** | Amazon Athena, Amazon Kinesis, Amazon OpenSearch Service |
| **Application Integration** | AWS AppSync, Amazon EventBridge, Amazon SNS, Amazon SQS, AWS Step Functions |
| **Compute** | Amazon EC2, AWS Elastic Beanstalk, AWS Lambda |
| **Containers** | Amazon ECR, Amazon ECS, Amazon EKS |
| **Database** | Amazon Aurora, Amazon DynamoDB, Amazon ElastiCache, Amazon RDS |
| **Developer Tools** | AWS Amplify, AWS CloudShell, AWS CodeArtifact, AWS CodeBuild, AWS CodeDeploy, AWS CodePipeline, AWS X-Ray, Amazon Q Developer |
| **Management & Governance** | AWS AppConfig, AWS CDK, AWS CloudFormation, AWS CloudTrail, Amazon CloudWatch, AWS CLI, AWS Systems Manager |
| **Networking & Content Delivery** | Amazon API Gateway, Amazon CloudFront, ELB, Amazon Route 53, Amazon VPC |
| **Security, Identity & Compliance** | Amazon Cognito, IAM, AWS KMS, AWS Secrets Manager, AWS STS, AWS WAF |
| **Storage** | Amazon EBS, Amazon EFS, Amazon S3 |

{{% notice info %}}
Kỳ thi sử dụng tên viết tắt chính thức. Ví dụ: "Amazon SNS" thay vì "Amazon Simple Notification Service".
{{% /notice %}}
